# 50-Small-Web-Projects
Repositorio de mini-proyectos web desarrollados como práctica a partir del curso “50 Projects in 50 Days - HTML, CSS and JavaScript” de Udemy.

## Descripción

Este repositorio recopila una serie de pequeños ejercicios interactivos creados siguiendo un curso de Udemy sobre desarrollo web con HTML, CSS y JavaScript. Cada carpeta contiene un mini-proyecto independiente, ideal para practicar conceptos básicos de frontend y experimentar con implementaciones sencillas reutilizables en otros desarrollos propios.

## Origen y créditos

Los proyectos incluidos NO han sido ideados ni desarrollados originalmente por mí: son implementaciones basadas en las clases y ejemplos del curso 50 Projects in 50 Days - HTML, CSS & JavaScript de Udemy (Brad Traversy & Florin Pop). Link al curso: https://www.udemy.com/course/50-projects-50-days/
Este repositorio tiene fines educativos y de práctica personal, buscando profundizar en técnicas frontend mediante la repetición y adaptación de sus retos.

## Proyectos incluidos

Algunos ejemplos de proyectos aquí disponibles:

- Barra de navegación interactiva con animaciones de iconos

- Formulario de login con efecto “wave”

- Botón estilizado con hover avanzado

- ...y otros componentes básicos reutilizables

Encontrarás la lista y descripción de cada proyecto en su propia carpeta.

## Cómo usar estos proyectos

Puedes clonar este repositorio o descargar los archivos para probarlos localmente.
Cada carpeta es un proyecto independiente; solo abre el archivo index.html correspondiente en tu navegador.

```bash

git clone https://github.com/Su-debug-glitch/50-Small-Web-Projects.git
cd 50-Small-Web-Projects
```

```bash

/
├─ Project 1_Expanding-Cards/        # Cartas que se expanden
├─ Project2_Progress-Steps/          # Barra de progreso
├─ Project3_Rotating-Navigation/     # Menú de navegación rotativo
└─ ...                               # Otros mini-proyectos
```
